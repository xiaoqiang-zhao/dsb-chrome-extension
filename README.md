# 代税宝浏览器插件

## 介绍

自动化报税，节省人力成本。RPA 机器人流程自动化。

## 需求描述

需要完成重亲与四川两地的自动报税，每个地区又分为一般纳税人和小规模纳税人两种，需要点击和填写。

重庆报税流程(图形验证码版)

四川报税流程(手机验证码版)
https://www.kdocs.cn/l/ceCgUVTmyzSi
